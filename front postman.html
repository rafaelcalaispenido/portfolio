Ajuda

Este painel funciona como um "Postman" simplificado: cada botão monta uma requisição e a envia diretamente do seu navegador para a API do eNotas, sem passar por nenhum backend próprio ou banco de dados intermediário. Nada do que você digita aqui fica salvo — se atualizar a página, os campos são limpos.
Passo a passo
1
Cole a API Key no campo no topo da tela (canto superior direito). Ela é obrigatória para qualquer chamada.
2
Escolha, no menu à esquerda, a operação que deseja executar (ex.: Consultar empresa, Emitir NFS-e, Baixar XML, etc).
3
Preencha os campos solicitados. Campos marcados com * são obrigatórios. Sempre que pedir "ID da Empresa", use o GW Empresa ID (veja abaixo onde encontrar).
4
Clique no botão de ação (Consultar, Listar, Emitir, Salvar, etc). A resposta da API aparece logo abaixo, em uma versão "amigável" traduzida para português e também no formato JSON técnico (clique em "Ver JSON" para alternar).
5
Em operações de inclusão/alteração (como "Incluir / Alterar Empresa", "Emitir NFS-e" e "Emitir NF-e"), o corpo da requisição é um JSON aberto. Você pode editar os valores dentro das chaves { }, mas não altere os nomes dos campos (as chaves, à esquerda dos dois-pontos) — eles precisam ser exatamente como a API do eNotas espera.
O que pode e o que não pode ser alterado
OK
Pode alterar: os valores dentro do JSON (CNPJ, razão social, valores, datas, descrições, etc) e os campos de formulário (IDs, datas de filtro, páginas).
!
Ponto de atenção: em geral, evite mudar os nomes das chaves do JSON (ex.: razaoSocial, cpfCnpj) e a estrutura/aninhamento dos objetos — isso costuma fazer a API rejeitar a requisição. A exceção são alguns campos que variam conforme a prefeitura (cada município pode nomear certos campos de forma diferente); nesses casos, siga o nome indicado na documentação/retorno daquela prefeitura específica.
!
Algumas alterações (ex.: dados de empresa) podem levar um tempo para refletir no Embed/Backoffice. Sempre consulte novamente após salvar para confirmar.
Onde encontrar as credenciais
KEY
API Key (Embutido): disponível em app-emissor.enotasgw.com.br/apikey
ID
GW Empresa ID (Embed): disponível em embed.enotas.com.br/backoffice
